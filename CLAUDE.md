# ИС Корпоративной Столовой — Экранные формы

## Обзор

Коллекция автономных HTML-прототипов (экранных форм) для информационной системы корпоративной столовой (АО «ТБанк»). Каждый файл — самодостаточная страница со встроенными стилями и скриптами.

## Технологии

- **Vanilla HTML + CSS + JavaScript** — без фреймворков, сборщиков и зависимостей
- **Canvas 2D API** — для графиков (без Chart.js)
- **CSV export** — через `Blob` + `URL.createObjectURL`
- Нет npm, package.json, tsconfig, бэкенда

## Модули

| Модуль | Директория | Описание |
|--------|-----------|----------|
| Авторизация | `авторизация/` | Пустая заготовка |
| Навигация | `модуль навигации/` | Пустая заготовка |
| План-меню | `план-меню/` | Просмотр и редактирование месячного плана меню |
| Доступные порции | `доступные порции/` | Карточки блюд для пользователя, мониторинг и редактирование порций для сотрудника |
| Идентификация | `модуль идентификации/` | СКДУ-система: идентификация по бейджу, журнал событий |
| Заказы | `модуль заказов/` | Панель работы с заказами, просмотр заказа посетителем, статистика, черновик заявки на закупку |
| Заявка на закупку | `Заявка на закупку/` | Создание и управление заявкой на закупку сырья |
| Отчёты | `отчеты/` | Отчёт по посещаемости и потреблению |

## Доменные сущности

`Сотрудник`, `Заказ`, `Позиция заказа`, `Статус заказа`, `Очередь заказов`, `Блюдо`, `Категория`, `Доступные порции`, `План-меню`, `Позиция план-меню`, `Сырьё`, `Заявка на закупку`, `Строка заявки`, `Договор поставки`, `Событие идентификации`, `СКУД терминал`, `Слот` (завтрак 08:30–10:30, обед 12:30–17:00).

## Соглашения

- **Дизайн**: желтый брендинг (`#FFDD2D`), карточные layouts, pill-бейджи, скруглённые углы, тени, sticky topbar
- **Демо-данные**: встроены прямо в JS каждой страницы
- **Тест-панели**: многие страницы содержат dev-панели для скриншотов/презентаций (внешний функционал)
- **Выходные на выходных**: модули учитывают, что в выходные/праздники данных нет
- **Статусы заказа**: `created` → `accepted` → `production` → `ready` → `issued` / `missed`
